# Projeto_IA_Saude_e_Bem-Estar

## 📌 Sobre o Projeto
O **Lumi** é um chatbot inteligente desenvolvido para auxiliar usuários com informações relacionadas à **saúde, bem-estar e qualidade de vida**.  

A aplicação utiliza **Inteligência Artificial** para responder perguntas, oferecer orientações básicas e promover hábitos saudáveis de forma acessível e interativa.

> ⚠️ Este projeto tem caráter **educacional e informativo**. Ele **não substitui orientação médica profissional**.

## 🎯 Objetivos
- Fornecer informações básicas sobre saúde
- Incentivar hábitos saudáveis
- Responder dúvidas comuns sobre bem-estar
- Demonstrar o uso de **IA em aplicações de saúde**
- Criar uma interface simples de interação com o usuário

---

## 🚀 Funcionalidades
- 🤖 Chatbot com IA
- 💬 Respostas automáticas sobre saúde e bem-estar
- 📚 Sugestões de hábitos saudáveis
- 🧠 Explicações simples sobre temas médicos
- 🔍 Sistema de perguntas e respostas
- 📱 Interface amigável
